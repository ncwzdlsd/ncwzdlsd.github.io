显然，这是一道拓补排序题。由于本蒟蒻拓补排序很不熟练，所以来练练手QwQ

前置芝士：拓补排序

考虑这几种情况：

  1. 可以找到一个正确的、稳定的拓补序列，输出 #
  2. 由于拓补序列是对于 DAG（有向无环图）而言的，所以如果图中存在环，就会出现矛盾输出 #
  3. 既不是情况 也不是情况 ，无法确定，输出 #

在输入的时候，每加一条边，我们就拓补排序一次。

如果输入的 中，，那么一定有环，直接结束。

如果全部进行完拓补排序后，可以找到一条长为 的序列，那么可以确定。

如果还有入度不为 的点，那么无符合的拓补序列，即第二种情况；
