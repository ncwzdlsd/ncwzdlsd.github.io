考试日会加上 * 标注。

寄。

20230302

考前几天都是晚二在机房不错，上 YbtOJ 打了个树状数组感觉良好，准备彻底放弃线段树了。晚上回家摆了摆，又是写作业写到 点多，从召集日前几天一直到春季测前一天我都是 点多睡的，初中同学居然跟我说我有黑眼圈了？不过上课确实感觉有点儿累。貌似有物理课，果然又睡了。跟老对儿说等他进羟基班了我老对儿会不会换成一个_____（这是不能说的话题，自己联想），他貌似有点儿伤心了（？，当然是开玩笑的哈哈我可不想老对儿是女生。体育课踢球了，感觉不错。跑操的时候又发生了一些奇奇怪怪的事情，乐。

20230303

主打的就是一个补觉。从第一节政治课一直在睡，英语课还被老师叫起来了但是我在睡觉所以根本不会（？，但是第五节物理课居然出乎意料的没睡着，幸亏没睡，要不最后老师让人去黑板上做题可能就找我了，压轴题不会，哭哭。中午去机房的时候一直在扯。下午上了个体育课踢球的时候一直在说“身价暴涨”这个梗，乐。地理课老师居然没找我要假期作业，开心.jpg。晚上试了试小黑屋的 Linux 感觉不错，被安利了 Geany 但是在考场上我后来还是选择了 CodeBlocks，虽然但是小黑屋的电脑是真的好，一体机 yyds！本来想着怎么也要把挂了好长时间的货车运输给调完结果最后也没调出来，一直在和 Kazdale 等人愉快地唠嗑。好像我第八节课就去了但是到晚饭的时候也没调出来。

吃完晚饭孙队有点儿 emo 让人不知所措，于是我们就和他一起把机房打扫了一下，有一说一机房的桌位里怎么全是垃圾啊，愤怒，八个人打扫了一个小时。走的时候从小黑屋借了一本书，孙队给我们发了 24OI 独家口罩然而考试那天我却忘带了也没找人去领，社恐.jpg。

后来回到教室一堆人跟我告别祝我考试顺利真的好感动啊 qwq，永远都爱十一班。单独拎出来一段表达我对十一班的感谢和爱🥰。

回家之后又摆了，洗了个澡想继续调货车运输但是内心一直不宁静然后莫名其妙就开始胸口疼，然后就开始胡思乱想突然就想到了今天是我在一年十一班的最后一天，又想到了我和这个班级的点点滴滴，然后就开始码字，写了一千多字和一年十一这个家的告别信，写到一半还给自己写哭了，等什么时候想起来了就发朋友圈里吧。大家分班考试加油啊！！！写完之后刷了会儿朋友圈，感觉整个考前状态不太对啊，明天应该要崩，所以十点多就准备睡觉了，睡前看了看博客上的板子，然后迷迷糊糊就睡着了。

*20230304

早上六点多就起了，比 gg 在群里发让吼的消息还早，但是我忘在群里吼了，悲。在车上看了看板子，眯了一会，然后就到了。过了一会儿就进去等了，大厅里好冷啊，冻手，所以后期一直都在暖手。看见别人了，有的打了个招呼，到后来看到他们都去了某个地方（？，没跟去，主要是怕万一考前跟别人唠嗑会大大影响我本就差得要死的考试状态。然后整个大厅里就只剩我一个蓝裤子和一堆别的学校和别的市的了，还有一个嘉汇的初中生在叫嚣什么初中就 Au，完蛋，要被初中生薄纱了。

进了考场，可以动电脑，第一反应就是打个 Tarjan，然后虽然我已经看了好多遍还是根本不会，悲。开考之后文件层层加密，从文件夹的 0309@thu 到题目 PDF 的 0304noi（可能记错了），FCC 是懂加密的。 遍历题干，感觉尚可。

先开 T1，一眼 普及-，然后数组开炸了报了一个我从未见过的编译错误，当时好多人因为中文路径没法编译监考老师在到处帮调，我也就趁机找老师问了一下为什么我编译失败结果老师说我数组开大了，感谢善良的监考老师要不我可能直接 T1 就挂了。打了一些部分分，二维数组能存的还有退化成一行的还有 的几个性质都打了，预估得分 ，还有一些部分分是后来改的，感觉还不错，好心情++。

再开 T2，打暴力，喜提 分，考完自测之后突然发现自己循环搞多了，人均 或 或 ，只有我这个小**得了 ，悲。

开 T3，打了个 分的排列组合，然后蒙上了 和 的解，然后就跑路了。

开 T4，打了个 的 分，然后不会了，跑了。

后来想随机挑一道题冲正解，离谱啊，根本不会。只能在四道题里反复横跳。最后把壁纸改成了”暴力出奇迹“，看来现在是暴力出奇寄。

差不多结束了，检查了一下文件夹，然后交了。交完之后也没跟人讨论，主要是自己太菜了不敢跟别人说话，在大连大学里吃了个饭就回家了。
