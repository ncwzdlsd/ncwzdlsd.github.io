20230311模拟赛

ncwzdlsd
  2023-03-11 14:42:48 2023-03-11 14:42 创建  
  • 总结 
 1.9k 字  7 分钟  

昨天听说今天又模拟赛，所以晚上本来想早睡，结果貌似还是摆烂摆到了十二点多。闹钟是随机铃声，结果今早给我整了个特别惊悚恐怖的铃声并且突然声音极大把我吓个半死/kk，于是我一怒之下把闹钟关掉继续呼呼大睡，然后后面三个闹钟好像就都没听到（？，等到醒的时候发现已经七点多了，于是火速收拾吃完早饭去机房。

一直想着今天不要再跪了，毕竟明天放假想要一个好心情啊 qwq。

开考之后，先开 T1，离谱的式子让我纠结了很长时间到底是什么意思，手推了一波样例结果自己的答案和样例输出不一样，心态–。看了一眼排行榜，发现 fjt 已经 T1 打出暴力砍掉 分了，完蛋，这波又要寄了。然后开始在草算纸上狂写，不知道为什么还是出了点问题，于是去开 T2。

看到 T2，题目名叫“桥”，那是什么，好一个 Tarjan，直接开摆乐！打了个全输出 的操作骗分，效果很好，拿到 分，不会爆零了，好耶 qwq。

不想在一道自己根本不会的题上浪费太多时间，于是火速润去 T3。想到了之前做的有关括号序列的题，先码出了一份 check 函数，但是很离谱啊居然把 check 写挂了，好像是入栈出栈的问题，寄。看了一眼数据范围，简单枚举了一下，发现 的情况貌似可以枚举做，因为合法序列种类很少，但是 check 函数写不出让我很难办啊，哭哭。鏖战 分钟之后选择跑路，嘿嘿。走之前打了个拼 RP 的骗分，豪赌一波 的 cout<<2，喜提 分，心态++。

看了一下榜，发现好多人已经切掉了 T1 或者打出了 T1 的暴力，所以想去再冲一波 T1，当一把 T1 暴力老哥复刻春季测逆天场景，又看了一下那个柿子：，又在草算纸上一通狂写突然明白了，不过那时候才明白确实挺智障的，然后打了个 KMP+ 暴力统计答案的思路交上去居然 分，不知道为什么挂了，后来发现是数组下标的问题，又是狂改，交上去 T 了。加了一个预处理，变成 分了，开始寻找大样例没找到，然后问老师才知道原来在题面上方的条条里，寄。测了一发大样例，寄了。把二维 char 改成 string 数组，调了调，又修改了一下数组下标的逻辑，交上去， 了，开心.jpg。但是第六个点 WA 掉了就很离谱，至今是未解之谜。

然后去看 T4，手造了几组数据发现有些点必须删除有些点可以随机删除，又因为删点要向下 DFS，所以大概率删除上方的点是更优的，所以直接豪赌一把把 的点都删掉就可以，这是好的。一交，没想到拿到了 分，开心。趁热打铁想再打一波 的解，但是不知道为什么一交上就 RE 了，离谱啊，所以我选择开溜。

然后开始纠结是去拼 T1 的正解还是把 T3 的 分暴力拿到，然后开始反复横跳并不断刷新排行榜发现自己的排名一组走低，哭哭。巅峰时期哥们儿直接全 Div. 3 Rank 2，到后来已经掉到 Rank 7,了。于是带上耳机开始听歌（顺便推一波自己的歌单 ，不知道从哪里搞得并且也听不懂但是感觉很适合敲代码的时候听诶 qaq，或者大家有什么好的歌单可以推荐给我诶 qwq）+摆烂+时不时狂打几行代码吓唬吓唬别人，结果最后突然发现 sr 居然把我反超了，寄。于是开始去 T3 骗分，又搞到 分，最后排名定格在 Rank 7。

总的来说这次模拟赛结果还是不错的，相比于上次有很大进步。抛开各种乱七八糟的事情，还是反映了自己的一些问题。最主要的就是 T1 正解总是打不出让人很难受啊，急需提高一下自己在 T1 这种思维/诈骗题上的能力的，以后应该多打打 CF 的 ABC 题了。最后 还行，至少明天可以愉快地摆烂啦 qwq。

最后说一点闲话，细数自己从 OI 生涯开始的所有模拟赛，从一开始逆天排序模拟赛莫名其妙挂大分只有 分，到后来春季测前模拟赛的逆天 ull 爆零，再到春季测的大寄垫底，再到前几天在 HL 的第一场模拟赛，可以说每一次的结果都是差强人意，但是冷静想想也是输得心服口服，自己的确存在很多的问题，无论是基础知识还是分析能力都需要提高。今天的模拟赛，可以说刚开始并没有什么太大的感觉，甚至还在担心自己会不会因为太想翻盘而影响心态，但是最后的结果反而还行，虽然在 T1 上还有一点小小的遗憾，毕竟考试时已经发现了 函数的取值规律可以说离正解只有一步之遥临门一脚了，但是最后还是没有打出正解，如果做出来了今天直接就起飞了 qaq。虽然但是，在 HL 的这两场模拟赛也让我认识到了人外有人天外有天，希望自己能不断进步越来越好吧。

所以明天我选择直接开摆

PS：cdx 说今天的模拟赛是选的 Div. 2 的 场模拟赛的 T1 诶，所以我不会是正常的 qaq。
