生物必修一第一二章知识点

发表于 2022-12-16
生物必修一第一二章知识点

  • 细胞基础

    • 细胞学说

      • “一切动植物都由细胞和细胞产物构成“

        • 细胞产物：激素、代谢物质等

    • 无细胞核的细胞

      • 骨髓造血干细胞形成的哺乳动物成熟的红细胞

      • 高等植物成熟的筛管细胞

    • 细胞是基本的生命系统

      • 以细胞代谢为基础的各种生理活动

      • 以细胞增殖、分化为基础的生长发育

      • 以细胞内基因的传递和变化为基础的遗传与变异

    • 生命系统的层次

      • 种群

        • 一定范围

        • 同种生物

        • 所有个体

        • 同一时间

      • 群落

        • 不同种群相互作用

        • 自然环境下，有直接/间接联系的各种生物集合

          • 例：菜市场的所有生物集合就不是群落

      • 生态系统

        • 群落+无机环境

        • 注意：无机环境也属于生命系统，属于生态系统中的无机环境

  • 细胞的多样性和统一性

    • 观察细胞

      • 显微镜的使用

        • 目镜与物镜

          • 目镜无螺纹，目镜越短，放大倍数越大

          • 物镜有螺纹，物镜越长，与玻片距离越近，放大倍数越大

        • 整理、收镜

          • 将显微镜的外表擦拭干净，使物镜偏向两旁，取下目镜放进镜头盒，反光镜与水平面垂直，将镜筒下降到最低处

        • 视野亮度调节

          • 调亮视野：大光圈、凹面镜

          • 调暗视野：小光圈、平面镜

      • 注意事项

        • 观察颜色深的材料，视野调亮
          观察颜色浅的材料，视野调暗

    • 原核细胞和真核细胞

      • 原核细胞

        • 结构特点

          • 没有染色体

          • DNA不与蛋白质结合，分布在拟核区

          • 细胞器只有核糖体（生产蛋白质、维持生命活动用）

          • 一般有细胞壁、细胞膜、细胞质
            （支原体无细胞壁）

        • 代表

          • 细菌、支原体
            注意真菌（蘑菇等）不是原核生物

      • 真核细胞

      • 性质

        • 多样性

          • 结构

          • 形态

          • 功能

        • 统一性

          • 基本结构：细胞膜、细胞质、核糖体

          • 化学组成

          • 遗传物质：所有有细胞结构的生物遗传物质均为DNA

      • 细胞壁成分对比

        • 原核生物：肽聚糖（支原体除外）

        • 植物：纤维素、果胶

        • 真菌：几丁质

  • 细胞组成

    • 糖类

      • 检验：斐林试剂（0.1g/mL的氢氧化钠溶液（甲液），0.05g/mL的硫酸铜溶液（乙液））

        • 注意事项

          • 实验材料

            • 还原糖含量高&&颜色较浅或接近白色

              • 举例：白萝卜匀浆、梨匀浆

          • 水浴加热（50~65℃）

            • 优点：受热均匀

          • 反应原理：还原糖将铜离子还原，生成砖红色沉淀氧化亚铜

          • 裴林试剂为蓝色

      • 组成元素：C、H、O

      • 种类

        • 单糖

          • 五碳糖

            • 核糖

            • 脱氧核糖

          • 六碳糖

            • 葡萄糖（都有）

            • 果糖（植物细胞中）

            • 半乳糖（动物细胞中）

        • 二糖

          • 麦芽糖：两分子葡萄糖

          • 蔗糖：一分子果糖+一分子葡萄糖

          • 乳糖：一分子葡萄糖+一分子半乳糖

          • 注意：二糖=对应单糖×2-一分子水

        • 多糖（单糖排列方式不同）

          • 淀粉（植物细胞中）

          • 糖原

            • 肝糖原

              • 调节血糖&&储能

            • 肌糖原

              • 储能

          • 纤维素

          • 几丁质

            • 应用

              • 能与溶液中的重金属离子结合，废水处理

              • 食品包装纸、食品添加剂

              • 人造皮肤

          • 膳食纤维：“第七类营养素”，属于多糖

            • 作用：促进肠胃蠕动排空

          非还原糖：蔗糖、淀粉、糖原

      • 功能

        • 能源物质（不都是）

        • 结构成分：细胞壁等

        • 细胞通讯、识别作用的基础，细胞膜的组成成分

          • 糖蛋白：识别作用

    • 脂质

      • 组成元素：C、H、O，部分有P

      • 特点

        • 氧含量低于糖类，碳、氢含量高于糖类

          • 脂肪：C-75%，O-13%

          • 糖类：C-44%，O-50%

          有机物能量在C-H键中，脂肪中C-H多

          脂肪多种子要浅种：分解时耗氧量大

        • 通常不溶于水，溶于脂溶性物质（丙酮、乙醚、氯仿）

      • 分类

        • 脂肪

          • 检验：苏丹Ⅲ染液（0.01g/mL）

          • 注意事项

            • 用50%的酒精溶液洗去浮色（苏丹Ⅲ易溶于酒精）

            • 切片要薄：防止有的地方清晰，有的地方模糊

            • 花生种子要提前浸泡，便于切成薄片，
              但不宜过长，防止薄片变形

          • 功能

            • 储能物质

            • 绝热体（保温）

            • 缓冲&&减压，保护内脏器官

          • 结构特点

            • 三分子脂肪酸、一分子甘油（三酰甘油、甘油三酯）（非多聚体）

            • 脂肪酸：碳原子以单键连接，饱和，固态；以双键连接，不饱和，液态

            • 非极性，不溶于水

        • 磷脂

          • 功能

            • 结构脂质

          • 元素组成

            • C、H、O、N、P

        • 固醇

          • 胆固醇

            • 构成动物细胞膜

            • 脂质运输

          • 性激素

          • 维生素D

            • 促进肠道对Ca和P的吸收

            • 人皮肤表面的胆固醇光照可转化为维生素D

    • 蛋白质

      • 检验：双缩脲试剂（0.1g/mL的NaOH溶液（A液），0.01g/mL的$CuSO_4$溶液（B液））

        • 反应原理：$Cu^{2+}$与肽键反应生成紫色络合物

      • 注意事项

        • 若用大豆，需提前浸泡

      • 生命活动的主要承担者

      • 功能

        • 结构蛋白

        • 功能蛋白

          • 催化作用

            • 酶（绝大多数，核酶不是蛋白质）

          • 运输作用

            • 血红蛋白：运输氧气和$CO_2$

          • 调节作用

            • 胰岛素：调节血糖

            胰岛素不能口服：被消化液分解

            • 生长激素

          • 免疫作用

            • 抗体（免疫球蛋白）

      • 基本组成单位——氨基酸

        • 结构

          • 至少一个氨基（$-NH_2$）和一个羧基（-COOH）

          • 至少一组氨基羧基连在同一个碳原子（中心碳原子）上，且中心碳原子连接一个H原子和R基

      • 结构多样性

        • 结合方式

          • 脱水缩合

        • 肽键

          • CONH连在一起

          • 数氨基酸个数时数肽键

        • 氨基酸→多肽→+空间结构→蛋白质

        • 结构多样性的原因（结构的角度分析）

          • 氨基酸种类、数量、排列顺序不同

          • 肽链空间结构不同

          根本原因：基因的多样性

      • 变性（不可逆）

        • 本质：空间构向被破坏，导致理化性质改变和生物活性丧失

        • 方式

          • 加热

          • 加酸加碱

          • 加酒精

      • 盐析（可逆）

        • 溶液溶解度变化导致蛋白质析出

      酶切断肽键
      加热不破坏肽键

      • 应用

        • 用热水洗衣服：防止高温使蛋白质变性凝固难以洗涤

        • 用70%酒精杀菌：渗透性强，使细菌蛋白质凝固

      • 蛋白质计算

        • 肽键数（脱去水分子数）=氨基酸数-肽链数（+R基中肽键数）

        • 氧原子数=肽键数+2×肽链数（+R基中氧原子数）=氨基酸中氧原子总数-脱去水分子数

        • 蛋白质相对分子质量=氨基酸数×蛋白质平均相对分子质量-脱去水分子数×18

        环肽：肽键数=脱去水分子数=氨基酸数

        • 二硫键：脱去2个氢原子

    • 无机物

      • 水

        • 含量：60%~95%

        • 作用

          • 细胞内良好的溶剂

          • 参与细胞内的生物化学反应

          • 提供液体环境

          • 运送营养物资和细胞新陈代谢中产生的废物

        • 水分子的特点

          • 极性：易与带有正负电荷的分子（或离子）结合

          • 流动性：氢键的存在

          • 比热容高（氢键的存在）：温度不易发生改变，便于维持生命系统的稳定性

        • 存在形式

          • 自由水：溶剂

          • 结合水：与蛋白质、多糖等结合

            • 结合水吸水能力：淀粉>蛋白质>脂肪

        • 自由水比例越大，细胞的代谢就越旺盛；结合水越多，细胞抵抗不良环境的能力就越强，抗逆性越强

      • 无机盐

        • 含量：1%~1.5%（占鲜重）

        • 存在形式

          • 大多是以离子形式

          • 少数以化合物形式

            • 碳酸钙

            • 羟基磷酸钙

        • 生理功能

          • 组成化合物

            • Mg构成叶绿素

            • Fe构成血红素

            • P组成细胞膜、细胞核

            • I组成甲状腺激素

          • 维持生命活动

            • 缺钠离子：神经、肌肉细胞兴奋性较低，引发肌肉酸痛、无力

            • 缺钙离子（钙离子可降低肌肉兴奋性）：抽搐

          • 维持平衡

            • 酸碱平衡：碳酸

            • 渗透压平衡：钠离子、氯离子

        • 植物必须无机盐的判断

          • 对照实验

            • 空白对照

            • 自身对照

          • 注意事项

            • “生长状况一致的”

            • “相同且适宜的条件”

    • 核酸

      • 种类

        • 脱氧核糖核酸（DNA）

          • 分布：细胞核、线粒体、叶绿体

        • 核糖核酸（RNA）

          • 分布：细胞质

          RNA也有催化作用

      • 基本组成单位——核苷酸

        • 组成

          • 一分子含氮碱基（A、T（脱氧核苷酸独有）、C、U（核糖核苷酸独有）、G）

          • 一分子五碳糖

          • 一分子磷酸

        • 分类

          • 核糖核苷酸

          • 脱氧核糖核苷酸

        • 结构

          • RNA：单链

          • DNA：平行反向的双螺旋

        • 多样性

          • 排列顺序、数量的多样性

        • 功能

          • 细胞内携带遗传信息的物质

          • 控制蛋白质的生物合成

        病毒只含一种核酸，朊病毒没有核酸，通过蛋白质作为遗传物质

        • 生物大分子

          • 骨架：碳链

          • 基本单位：单体

          • 水解产物

          蛋白质初步水解：多肽（或短肽）
          脂肪初步/完全水解：甘油、脂肪酸

          多聚体的特征
          1.具有复杂的空间结构
          2.靠非共价键（氢键）（共用电子对）维持结构
          3.有方向性
